# 🍫 All Things Store

Landing page de uma loja de chocolates artesanais, desenvolvida com HTML, CSS e JavaScript puros.

## 🔗 Demo

[Ver site online](https://seu-usuario.github.io/all-things-store/)

## ✨ Funcionalidades

- Design responsivo (mobile, tablet, desktop)
- Menu hambúrguer no mobile
- Carrinho de compras com contador dinâmico
- Notificações toast ao adicionar produtos
- Scroll suave entre seções
- Acessibilidade (ARIA labels, navegação por teclado)

## 🛠️ Tecnologias

- HTML5 semântico
- CSS3 (Grid, Flexbox, Custom Properties)
- JavaScript vanilla (ES6+)

## 🚀 Como rodar localmente

```bash
git clone https://github.com/seu-usuario/all-things-store.git
cd all-things-store
```

Depois é só abrir o `index.html` no navegador (ou usar a extensão Live Server do VSCode).

## 📁 Estrutura

```
all-things-store/
├── index.html
├── css/style.css
├── js/script.js
└── assets/img/
```
